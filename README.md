## Requirements

- Node.js
- npm

## Installation

1. Clone this repository:
    ```bash
    git clone <REPOSITORY_URL>
    ```

2. Navigate to the project directory:
    ```bash
    cd <PROJECT_FOLDER_NAME>
    ```

3. Install the dependencies:
    ```bash
    npm install
    ```

## Running the Application

To start the application in development mode:

```bash
npm run dev
